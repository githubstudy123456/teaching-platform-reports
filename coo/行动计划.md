## 一、内容优先级排序

### 1.1 学科建设顺序

| 优先级 | 学科 | 核心理由 |
|--------|------|----------|
| **第1批** | 数学、物理、化学 | 已有内容基础好（物理68章/化学19章/数学89章），仿真需求强，3D模型资源丰富（PhET/PubChem） |
| **第2批** | 生物、地理 | 3D模型资源较丰富（NASA/Sketchfab），适合3D展示，内容制作难度中等 |
| **第3批** | 语文、英语 | 现有内容基础薄弱，但制作难度低（主要是图文/音视频），无需复杂仿真 |
| **第4批** | 历史、信息技术、政治 | 内容偏图文，政治/历史变动少，信息技术可复用开源编程可视化 |

### 1.2 详细顺序及理由

```
1. 数学（第1批）— 88章内容基础，GeoGebra iframe免费商用，3D立体几何可自建模
2. 物理（第1批）— 66章内容基础，PhET仿真覆盖力学/电磁学/光学，3D仪器 Blender 建模简单
3. 化学（第1批）— 17章内容基础，3Dmol.js+PubChem分子结构免费接入，实验仪器3D需求明确
4. 生物（第2批）— 细胞/器官/微生物3D模型 Sketchfab 有免费资源，图文制作难度低
5. 地理（第2批）— NASA 3D地形/行星模型免费商用，ECharts交互地图
6. 语文（第3批）— 仅1章内容缺口大，以图文/音频形成为主，无需仿真，制作难度低
7. 英语（第3批）— 仅1章内容缺口大，听力/阅读内容制作难度低
8. 历史（第4批）— 文物3D Sketchfab 免费区，时空轴组件可复用
9. 信息技术（第4批）— 代码编辑器/算法可视化有成熟开源组件（Monaco Editor）
10. 政治（第4批）— 内容稳定，图文形成为主，ECharts数据可视化
```

## 三、新建服务器后要做的事清单

### 服务器购买后第一步

```bash
# 1. SSH 登录服务器
ssh root@your-server-ip

# 2. 创建普通用户（禁止root直接登录）
adduser ubuntu && usermod -aG sudo ubuntu

# 3. SSH密钥配置（禁用密码登录）
mkdir -p ~/.ssh && chmod 700 ~/.ssh
cp /root/.ssh/authorized_keys /home/ubuntu/.ssh/
chown ubuntu:ubuntu ~/.ssh/authorized_keys
# 编辑 /etc/ssh/sshd_config 设置 PasswordAuthentication no

# 4. 系统更新
apt update && apt upgrade -y

# 5. 安装基础工具
apt install -y curl wget git vim htop unzip
```

### 域名解析步骤

```
1. 购买域名（阿里云/腾讯云/Cloudflare）
2. 进入域名控制台 → DNS解析
3. 添加记录：
   - A记录：@ → 服务器IP
   - A记录：www → 服务器IP
   - CNAME：api → @ (或ALB地址)
   - CNAME：res → CDN域名
4. 等待生效（通常5分钟-2小时）
```

### SSL证书步骤（Let's Encrypt）

```bash
# 安装 certbot
apt install -y certbot python3-certbot-nginx

# 申请证书（需域名已解析）
certbot --nginx -d yourdomain.com -d api.yourdomain.com

# 自动续期测试
certbot renew --dry-run

# 添加 crontab 自动续期
crontab -e
# 添加：0 0 * * * certbot renew --quiet
```

### Docker 部署步骤

```bash
# 1. 安装 Docker
curl -fsSL https://get.docker.com | sh
usermod -aG docker ubuntu

# 2. 安装 Docker Compose
curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
chmod +x /usr/local/bin/docker-compose

# 3. 创建项目目录
mkdir -p /opt/teaching-platform
cd /opt/teaching-platform

# 4. 克隆代码
git clone https://github.com/your-org/teaching-platform.git .

# 5. 配置环境变量
cp .env.example .env
vim .env  # 填写数据库密码、密钥等

# 6. 启动服务
docker-compose up -d --build

# 7. 查看日志
docker-compose logs -f
```

### 数据库初始化步骤

```bash
# PostgreSQL 初始化
docker-compose exec db psql -U postgres -d teachplatform -f /docker-entrypoint-initdb.d/init.sql

# Redis 配置验证
docker-compose exec cache redis-cli ping
```

# 仿真实验平台调研报告

> 调研时间：2026年6月24日
> 调研方式：curl 访问各平台官网获取公开信息

## 1. PhET Interactive Simulations

| 项目 | 详情 |
|------|------|
| **网址** | https://phet.colorado.edu |
| **覆盖学科** | 数学、物理、化学、生物、地球科学 |
| **内容形态** | 互动仿真（HTML5/JavaScript） |
| **授权方式** | **免费商用**（Creative Commons Attribution许可，源代码开源） |
| **接入方式** | iframe 嵌入、直接下载、本地部署 |
| **优缺点点评** | ✅ **优点**：全球最知名的K-12理科仿真平台，由诺贝尔奖得主Carl Wieman创立，基于大量教育研究设计，内容质量高，支持50+语言包括中文，完全免费可商用。<br>❌ **缺点**：界面为英文为主（虽然有中文翻译但原生中文内容少），主要覆盖理科基础领域，语文/英语/地理等人文社科内容缺失，不提供完整课程体系。|

## 2. NB诺老师 (NOBOOK)

| 项目 | 详情 |
|------|------|
| **网址** | https://www.nobook.com |
| **覆盖学科** | 物理、化学、生物、小学科学 |
| **内容形态** | 3D模型、互动仿真、图文 |
| **授权方式** | **付费**（提供学校/机构授权、个人订阅） |
| **接入方式** | Web端直接使用、小程序、APP（有合作对接平台 open.nobook.com） |
| **优缺点点评** | ✅ **优点**：国内领先的虚拟实验平台，专注于K-12理科实验，3D视觉效果出色，被央视等权威媒体报道过，与国内教材同步度高，支持实验测评功能。<br>❌ **缺点**：付费模式，商业授权费用较高；主要覆盖理科实验，语数英等主科内容缺失；非开源，无法自行二次开发或免费部署。|

## 6. GeoGebra

| 项目 | 详情 |
|------|------|
| **网址** | https://www.geogebra.org |
| **覆盖学科** | 数学（代数、几何、三角函数、微积分、概率统计等） |
| **内容形态** | 互动仿真、3D模型、动态几何 |
| **授权方式** | **免费商用**（GPL v3 开源，核心引擎开源） |
| **接入方式** | iframe 嵌入、直接引用（提供 Embed API）、GeoGebra Classic 桌面版、Calculator Suite网页版 |
| **优缺点点评** | ✅ **优点**：全球最流行的动态数学软件之一，完全免费开源，支持嵌入到任何网站，数百万教师贡献的免费互动资源库（1M+），覆盖数学各分支，API文档完善。<br>❌ **缺点**：仅覆盖数学学科，不适合物理化学等实验仿真；界面有一定学习成本；主要面向4-12年级，高等数学资源相对较少；服务器在海外，国内访问可能较慢。|

## 总结与建议

1. **如果需要免费可商用的理科仿真**：首选 **PhET**（国际领先，完全免费，支持嵌入）或 **GeoGebra**（数学专项，开源免费）
2. **如果需要国内K-12理科实验且预算充足**：**NB诺老师** 是最佳选择，3D效果出色
3. **如果需要数学互动教学工具**：**GeoGebra** 是行业标准，资源丰富
4. **如果需要综合资源而非仿真**：**学科网** 和 **洋葱学园** 可作为内容补充
5. **公立学校场景**：优先考虑 **国家智慧教育平台**，权威免费

> ⚠️ 注：以上信息基于2026年6月各平台官网公开信息整理，授权方式和接入方式可能随平台更新变化，建议对接前与平台方确认最新商务政策。

# 3D模型库调研报告

> 调研时间：2025年
> 调研目的：为教学平台筛选可免费商用的3D模型资源

---

## 一、国际3D模型库平台

### 1. Sketchfab（重点：免费区）

| 项目 | 详情 |
|------|------|
| **平台/名称** | Sketchfab (https://sketchfab.com) |
| **模型格式** | `.glb`, `.gltf`, `.obj`, `.fbx`, `.usdz` |
| **授权方式** | 部分免费（需查看具体License）；免费模型多采用 CC Attribution / CC Attribution-NonCommercial 授权；付费模型价格不等 |
| **学科覆盖** | 广泛——动物、植物、科学仪器、历史文物、地理景观等 |
| **获取方式** | API接口：`https://api.sketchfab.com/v3/models?free=true`；可直接下载（部分需注册） |

**免费模型示例：**
- API返回的免费模型包含教育类素材
- 需注意：并非所有Sketchfab模型都可商用，部分为CC-NC或更严格限制
- 下载格式优先选 `.glb`（Web兼容性最佳）

---

### 2. 3D Warehouse（SketchUp 3D Warehouse）

| 项目 | 详情 |
|------|------|
| **平台/名称** | 3D Warehouse (https://3dwarehouse.sketchup.com) |
| **模型格式** | `.skp` (SketchUp原生格式)，部分支持 `.obj`, `.3ds`, `.dae` 导出 |
| **授权方式** | 用户上传，授权各异；部分免费，部分需购买；无统一授权体系 |
| **学科覆盖** | 建筑、机械、人物、家具、车辆等；物理仪器类模型较少 |
| **获取方式** | 直接下载（无需注册）；搜索功能基于网页界面 |

**注意：** 模型质量参差不齐，教学用需筛选。

---

### 3. NASA 3D Resources

| 项目 | 详情 |
|------|------|
| **平台/名称** | NASA Science 3D Resources (https://science.nasa.gov/3d-resources/) |
| **模型格式** | `.obj`, `.gltf`, `.glb`, `.stl`, `.usdz` |
| **授权方式** | **免费商用**——NASA素材属于公共领域（Public Domain） |
| **学科覆盖** | 天文（行星、卫星、小行星）、航天器、星座、地理地形 |
| **获取方式** | 直接下载，无需注册；部分通过3D打印友好格式提供 |

**典型模型：**
- 月球、火星、地球3D地形
- 各星球高清纹理模型
- 航天器3D模型（卫星、探测器等）

---

### 4. PubChem 3D（化学分子）

| 项目 | 详情 |
|------|------|
| **平台/名称** | PubChem (https://pubchem.ncbi.nlm.nih.gov) |
| **模型格式** | `.pdb`, `.sdf`, `.mol`, `.json`（3D坐标数据） |
| **授权方式** | **免费商用**——NIH/NLM公共数据 |
| **学科覆盖** | 化学分子结构、药物小分子 |
| **获取方式** | REST API：`https://pubchem.ncbi.nlm.nih.gov/rest/pug/compound/name/{name}/record/JSON?record_type=3d` |

**API示例（获取咖啡因3D结构）：**
```
https://pubchem.ncbi.nlm.nih.gov/rest/pug/compound/name/caffeine/property/MolecularFormula,CanonicalSMILES/JSON
```
返回分子式和SMILES结构式。

## 三、学科典型3D模型需求分析

### 1. 物理学科——典型3D模型

| 类别 | 典型模型 | 免费资源情况 |
|------|----------|--------------|
| **力学** | 滑轮、滑轮组、斜面、木块、杠杆、天平 | ❌ 3D Warehouse有零星素材；大多需自行建模 |
| **运动学** | 匀速运动小车、弹簧振子、单摆 | ❌ 无成熟资源库 |
| **电磁学** | 电路元件（电阻、电容、电感）、磁铁、线圈 | ❌ 无成熟资源库 |
| **光学** | 凸透镜、凹透镜、平面镜、三棱镜 | ❌ 无成熟资源库 |
| **热学** | 分子运动模型、气体钢瓶 | ❌ 无成熟资源库 |
| **声学** | 音叉、波形演示 | ❌ 无成熟资源库 |

**物理学科建模建议：**
- 简单几何体（斜面、木块）→ 可用 SketchUp / Blender 快速建模
- 复杂仪器（电路元件）→ 建议用 Blender 或商业3D软件制作
- 透镜、棱镜 → 几何结构简单，可自行建模

---

### 2. 化学学科——典型3D模型

| 类别 | 典型模型 | 免费资源情况 |
|------|----------|--------------|
| **分子结构** | 水分子(H₂O)、二氧化碳(CO₂)、苯环等 | ✅ **PubChem** 提供完整3D坐标 |
| **有机化合物** | 甲烷、乙醇、丙酮、葡萄糖 | ✅ PubChem |
| **实验仪器** | 烧杯、试管、锥形瓶、酒精灯 | ⚠️ Sketchfab有零星素材 |
| **晶体结构** | NaCl晶格、金刚石结构 | ❌ 无专项资源库 |
| **反应装置** | 蒸馏装置、滴定管 | ⚠️ Sketchfab有零星素材 |

**化学学科建议：**
- 分子结构 → **优先使用 PubChem**（免费、可商用、格式为 `.pdb/.sdf`）
- 实验仪器 → Sketchfab免费区或自行建模

---

### 3. 生物学科——典型3D模型

| 类别 | 典型模型 | 免费资源情况 |
|------|----------|--------------|
| **细胞结构** | 动物细胞、植物细胞、细菌 | ⚠️ Sketchfab有零星素材 |
| **细胞器** | 线粒体、叶绿体、细胞核 | ⚠️ Sketchfab有零星素材 |
| **人体器官** | 心脏、大脑、肾脏等 | ⚠️ Sketchfab/3D Warehouse有 |
| **微生物** | 病毒（冠状病毒）、细菌 | ⚠️ Sketchfab有 |
| **植物** | 花、叶、根 | ⚠️ Sketchfab有 |

**生物学科建议：**
- Sketchfab免费区搜索 `cell`, `organ`, `virus` 等关键词
- NASA地形模型可用于地理教学

---

### 4. 地理/地球科学——典型3D模型

| 类别 | 典型模型 | 免费资源情况 |
|------|----------|--------------|
| **地形模型** | 山脉、河流、峡谷 | ✅ **NASA 3D Resources** 提供地球/月球/火星地形 |
| **天文模型** | 行星、卫星、星座 | ✅ NASA 3D Resources |
| **地图地形** | 3D地球仪、地形起伏 | ⚠️ Sketchfab有零星素材 |

## 四、模型格式说明与选用建议

| 格式 | 适用场景 | Web兼容性 | 教学平台建议 |
|------|----------|----------|--------------|
| **.glb** | Web端3D展示（最推荐） | ✅ 最佳 | 首选格式 |
| **.gltf** | Web端3D展示 | ✅ 最佳 | 备选格式 |
| **.obj** | 通用3D格式 | ✅ 良好 | 次选格式 |
| **.pdb** | 分子结构 | ⚠️ 需转换 | 化学生物专用 |
| **.sdf** | 分子结构 | ⚠️ 需转换 | 化学生物专用 |
| **.skp** | SketchUp | ❌ 需转换 | 不推荐直接使用 |
| **.stl** | 3D打印 | ⚠️ 需转换 | 仅3D打印场景 |

## 五、总结与建议

### 可直接使用的免费资源

| 学科 | 推荐资源 | 获取方式 |
|------|----------|----------|
| **化学（分子）** | PubChem 3D | API或网页下载 `.pdb/.sdf` |
| **天文/地理** | NASA 3D Resources | 直接下载 `.glb/.obj` |
| **通用3D** | Sketchfab免费区 | API搜索，筛选 `isDownloadable=true` |

### 需要自行建模的资源

| 学科 | 模型类型 | 建模工具建议 |
|------|----------|--------------|
| **物理** | 滑轮、斜面、电路、透镜、磁铁 | Blender（开源）、SketchUp |
| **化学** | 实验仪器（烧杯、试管） | Blender |
| **生物** | 细胞、器官、微生物（精细） | Blender、Maya |

### 建模工作量评估

| 难度 | 模型类型 | 预计工时 |
|------|----------|----------|
| ⭐ 简单 | 斜面、木块、透镜、棱镜 | 5-15分钟/个 |
| ⭐⭐ 中等 | 滑轮组、电路元件、烧杯 | 15-30分钟/个 |
| ⭐⭐⭐ 复杂 | 细胞结构、人体器官、分子晶体 | 30分钟-2小时/个 |
